# OBB 标注工具选择与人工界面规则计划

## 背景

用户没有现成标注工具，需要为 16 张 YOLO/OBB 冒烟样本选择一个轻量、本地、能导出 Ultralytics OBB 标签的标注方案。

同时，用户提出新的长期规则：

- 需要人工填写的内容必须去掉不必要信息。
- 图像排列方式必须优先考虑人工查看是否方便。

该规则需要写入 `rules/` 下的独立规则文件，后续所有人工复核包、标注包和表格都应遵守。

## 目标

1. 调研本地 OBB 标注工具。
2. 选择一个最适合当前项目的工具。
3. 形成具体安装/使用流程。
4. 新增人工界面/表格简化规则文件。
5. 如有必要，调整本地 `smoke_review_index.html` 的后续生成原则。

## 选择标准

优先级从高到低：

1. 能本地运行，不上传图纸。
2. 支持旋转框/四点多边形标注。
3. 能导出或容易转换为 Ultralytics OBB 格式。
4. 安装和使用足够简单，适合当前 16 张冒烟样本。
5. 界面适合查看机械图纸大图。
6. 后续可扩展到 83 张样本。

## 本轮不做

- 不开始真实标注。
- 不训练 YOLO/OBB。
- 不上传任何图纸。
- 不安装工具，除非用户确认。

# ISAT 标注工具调研计划

## 背景

用户反馈：有相关评论认为 ISAT 比 Labelme 更好用。当前项目已推荐 `Labelme + 项目内转换脚本`，但在用户开始标注前，应重新调研 ISAT 是否更适合 YOLO/OBB 标题栏冒烟标注。

## 目标

判断是否应将当前推荐工具从 Labelme 调整为 ISAT，或保留 Labelme。

重点比较：

1. 是否能本地运行，不上传图纸。
2. 是否支持 polygon/旋转框等适合 OBB 的标注方式。
3. 是否能导出 YOLO/OBB，或是否容易转换。
4. 是否适合 16 张冒烟样本的轻量标注。
5. 是否比 Labelme 更方便查看大图和快速标注。
6. 安装复杂度和长期维护风险。

## 本轮不做

- 不安装 ISAT。
- 不开始标注。
- 不训练 YOLO/OBB。
- 不删除 Labelme 方案。

## 输出

- 新增 ISAT 与 Labelme/CVAT 对比调研文档。
- 若 ISAT 更合适，更新标注工作流文档。
- 若 Labelme 仍更合适，记录保留理由。
- 更新 RPD 和 TODO。
